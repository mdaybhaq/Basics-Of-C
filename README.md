## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/mdaybhaq/Basics-Of-C.git
```

2. Open the project folder:

```bash
cd Basics-Of-C
```

3. Compile a program:

```bash
gcc filename.c -o output
```

4. Run the program:

```bash
./output
```
